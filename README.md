1. **Create Account** - Start by creating your first account
2. **Deposit Money** - Add funds using your Account ID
3. **Withdraw Money** - Remove funds with balance checking
4. **View Account** - Check account details with ID
5. **Transaction History** - See complete transaction records

## Account Structure

Each account contains:
- Unique 8-digit ID (auto-generated)
- Name
- Gmail address
- Password
- Balance
- Transaction history
